# Data Analysis Skill

Professional data analysis toolkit for comprehensive data exploration, cleaning, visualization, and statistical analysis.

## Quick Start

1. **Analyze data automatically**:
   ```bash
   python scripts/auto_analysis.py your_data.csv
   ```

2. **Generate data profile**:
   ```bash
   python scripts/profile_data.py your_data.csv
   ```

## Features

- **Data Profiling**: Quick quality assessment and statistics
- **Data Cleaning**: Missing values, outliers, duplicates, type conversion
- **Visualization**: 20+ chart templates for distribution, comparison, relationships
- **Statistical Analysis**: Hypothesis testing, correlation, regression
- **Machine Learning**: Classification, regression, clustering, dimensionality reduction
- **Automated Reporting**: HTML reports with visualizations

## Supported Formats

- CSV
- Excel (.xlsx, .xls)
- JSON
- SQL databases (via connection)
